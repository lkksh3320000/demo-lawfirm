## 一、v2 相對於 v1 的異動摘要

這一版最大的變化是：**曹律師本人首次直接提供了真實的行銷素材**（執業理念、完整案件類型、5則指標性案例、目標客群方向），取代了v1沿用前東家團隊頁推估的簡短版本。詳細決策記錄在 PM 文件 2.3 節。

| 項目 | v1 | v2 |
|------|----|----|
| 關於曹律師－經歷 | 具名「興望法律事務所實習律師／律師」 | **改為「七年豐富執業經驗」**（曹律師明確表示不具名前東家） |
| 服務項目分類 | 不動產／繼承／公司法人／一般民刑事（推估） | **民商事訴訟與非訟事件／刑事辯護與告訴代理／行政救濟／家事與繼承事件**（曹律師本人確認，新增「行政救濟」整類） |
| 指標性案例 | 無 | 新增5則真實案例摘要（金額用區間詞，標注待倫理規範確認） |
| 專業資格 | 無 | 新增：校園性別事件調查人才庫、職場霸凌調查人才庫調查員 |
| 目標客群 | 未定義 | 高資產客戶／企業主為主要目標，調性偏商務菁英＋安心可靠 |
| 首頁主視覺文案 | 「專業訴訟代理，陪您走過每一步法律程序」 | 「洞悉風險，精準佈局」，語氣更權威／商務 |
| 服務範本頁 | `services-real-estate.html`（不動產訴訟） | 改為 `services-civil-commercial.html`（民商事訴訟，曹律師較專長領域） |
| 配色 | 深藍＋金銅 | 不變（恰好與曹律師建議的海軍藍＋香檳金方向一致） |
