## 三、本版本的設計決策（想跟您討論的地方）

- **色彩**：深藍（`#13233b`）＋金銅（`#b08d57`），取「專業信賴」＋「質感點綴」的組合，可再調整
- **字體**：故意不用Google Fonts等外部CDN，只吃系統內建中文字體（Noto Sans/Serif TC、微軟正黑體等fallback），確保離線也能正常顯示、不多一個外部依賴
- **服務項目頁**：4頁共用同一套模板，本版本只做「不動產訴訟」1頁完整示範，確認滿意後v2再套用到其餘3頁，避免改版型要重改4份
- **後台**：只做「AI草稿審核」這個PG文件裡最核心的把關機制，會員管理／諮詢表單紀錄後台先留佔位連結（點了會跳提示，不是壞掉，是刻意標示為v2後續規劃）
