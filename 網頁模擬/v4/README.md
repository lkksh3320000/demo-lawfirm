| 導覽列 | 首頁/關於/服務 | 新增「**最新消息**」，變成首頁/關於/服務/最新消息 |
| 最新消息頁 | 無 | **新增** `news.html`，收錄曹律師受邀演講2則紀錄（其中一則含真實活動照片） |

服務範圍的**文字內容本身**（民事6類/刑事3類/家事3類細項）與v3完全相同，本輪純粹是**排版呈現方式**的調整，非內容變更。
